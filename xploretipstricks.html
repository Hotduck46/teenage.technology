Action Shortcut
Open/Close Settings: Option Comma

Share Sheet: Option Shift S
Reload page: Option R
Back: Option Left Arrow
Forward: Option Right Arrow
Return to search engine: Option S
Jump to web address bar Double click Space


Tabs:
To enable multiple tabs in Xplore, open Xplore and select View in the macOS menu bar, then Show Tab Bar.
